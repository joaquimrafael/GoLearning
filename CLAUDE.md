# GoLearning

Este projeto existe para **me ajudar a aprender Go do zero**. Não é um projeto de
produção — é um espaço de estudo onde vou incrementando anotações e código conforme
aprendo.

## Quem sou eu

- Estou começando em Go agora. Minha base principal é **C#** (linguagem que uso no
  trabalho atualmente).
- Não tenho preferência fixa de idioma — gosto de variar entre português e inglês.
- Aprendo melhor de forma incremental: pouco a pouco, um conceito de cada vez.
- Comparações com C# são bem-vindas para fixar conceitos novos.

## Como me ajudar

- **Não vá além do que eu já estudei.** Ao editar minhas anotações (ex.: `basic.md`),
  melhore só o que já está lá — não adicione tópicos novos que eu ainda não cobri.
- Quando eu pedir para aprender algo novo, **explique o conceito** antes de mostrar
  código, e prefira exemplos curtos e diretos.
- Sempre que introduzir um conceito, acompanhe com um **exemplo curto em Go**.
- Aponte boas práticas e idiomatismos de Go, mas sem me sobrecarregar — foque no
  que é relevante para o ponto atual do meu aprendizado.
- Se eu cometer um erro de entendimento nas anotações, me corrija explicando o porquê.

## Estrutura

- `basic.md` — minhas anotações de fundamentos, que vou ampliando aos poucos.

## Estilo das anotações

- Markdown com blocos de código usando syntax highlight (` ```go `, ` ```sh `).
- Português, com tom de anotação pessoal ("posso...", "declaro...").
- Mantenha o padrão existente ao adicionar conteúdo novo.
